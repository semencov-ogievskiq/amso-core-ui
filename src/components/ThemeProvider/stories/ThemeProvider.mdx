## Описание
 
Компонент ThemeProvider преднозначен для установки кастомной темы и глобальной кастомизации компонентов

## Базовый пример использования

Компонент является контекстом, по этому им нужно оборачивать все компоненты в приложение 

<Source code={`
<ThemeProvider theme={{}}/>
`}/>

## Установка темы

При установке темы не обязательно заполнять все поля формы. Незаполненые поля берутся из темы по умолчанию
