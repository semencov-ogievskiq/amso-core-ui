## Описание
 
Компонент Icon является заменой тега svg. Он требуется для интеграции иконок с библиотекой компонентов

## Базовый пример использования

Для использования компонента вставте в children содержимое своего svg и приведите код в jsx. Так же следует указать viewBox, так как в большенстве случаев у кажой иконки он свой.

## Установка цвета

Компонент Icon реализует props color с помощью которого можно выбрать цвет из темы
 
<Canvas of={Colors} />
<Controls of={Colors} />

Так же мы можем указать цвет явно через props fill. В таком случае props color не будет работать.

<Canvas of={Fill} />
<Controls of={Fill} />

Так же для компонента можно явно указать цвет через тему под конкретный выбранный props color, через настройку theme.components.Icon.Colors
Пример настройки:

<Source code={`
{
    [ICON_COLORS.SUCCESS]: "red",
}
`} />

## Установка размера

Компонент поддерживает указание размера иконки через props size

<Canvas of={Sizes} />
<Controls of={Sizes} />

Сам размер определяется темой через настройку theme.components.Icon.sizes.
Значение по умолчанию:

<Source code={`
{
    [ICON_SIZES.X_LARGE]: DIMENSION[32],
    [ICON_SIZES.LARGE]: DIMENSION[28],
    [ICON_SIZES.MEDIUM]: DIMENSION[24],
    [ICON_SIZES.SMALL]: DIMENSION[20],
    [ICON_SIZES.X_SMALL]: DIMENSION[16]
}
`} />
